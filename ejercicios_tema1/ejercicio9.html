1.¿En qué se diferencia JavaScript de Java?

  1. Propósito: JavaScript se utiliza principalmente para crear interacciones y comportamientos dinámicos en sitios web y aplicaciones web, mientras que Java se utiliza para crear aplicaciones de escritorio, aplicaciones móviles y otros tipos de software.
  2. Tipo de lenguaje: JavaScript es un lenguaje de programación interpretado, lo que significa que se ejecuta directamente en el navegador web del usuario final. Por otro lado, Java es un lenguaje de programación compilado, lo que significa que el código fuente se compila en bytecode que se puede ejecutar en una máquina virtual.
  3. Orientación a objetos: JavaScript es un lenguaje de programación orientado a objetos basado en prototipos, lo que significa que utiliza objetos prototipos para heredar y compartir funcionalidades entre objetos. Java, por otro lado, es un lenguaje de programación orientado a objetos basado en clases, lo que significa que se definen clases y se crean objetos a partir de ellas.
  4. Tipos de datos: JavaScript es un lenguaje de programación de tipado dinámico, lo que significa que los tipos de datos se asignan automáticamente en tiempo de ejecución. Java, por otro lado, es un lenguaje de programación de tipado estático, lo que significa que los tipos de datos se deben declarar explícitamente en tiempo de compilación.
